# DelftDashboard Watersheds Toolbox

A DelftDashboard toolbox for selecting and exporting watershed boundaries.

## Features

- Browse HydroBASINS and USGS WBD watershed datasets
- Select watersheds at multiple hierarchy levels
- Visualise watershed boundaries on the map
- Export selected watersheds with optional buffer

## Installation

```bash
pip install git+https://github.com/your-org/delftdashboard-watersheds-toolbox.git
```

Auto-discovered by DelftDashboard via entry points.
